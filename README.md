## Spring Security

1. 회원 정보를 저장할 테이블을 만들고 테이블과 연결할 도메인 만들기
2. 이 테이블과 연결할 엔티티 만들기
3. 회원 엔티티와 연결되어 데이터를 조회하게 해줄 레파지토리 만들기
4. 스프링 시큐리티에서 사용자 정보를 가져오는 서비스 만들기
